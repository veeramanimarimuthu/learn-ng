Property Binding

Property Binding is a single way data binding in which you can bind your class level properties to the components view. The syntax for property binding is: [attribute-name] = 'property-name' OR [ngModel] = 'property-name'


Event Binding

Event Binding provides you with a way to capture various types of events and execute some functionality if such events occur. The syntax for event binding is: (eventNameToBeCaptured) = "someFunction()" or (ngModelChange) = "someFunction()"

Two-Way Data Binding

Two way data binding can bind class level property with component view and any change from either end will be reflected on both the sides instantly. The syntax for two-way data binding is: [(attribute-name)] ='properties-name'
